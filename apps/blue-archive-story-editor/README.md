# 可视化剧情编辑器

## 紧急

- [ ] title (开头大标题)
- [ ] place (左上角地点标题)
- [ ] na (无人名对话)
- [ ] wait (整个剧情播放暂停)
- [ ] zmc (背景图放大聚焦)
- [ ] character (人物对话)
- [ ] characterEffect (人物特效)
  - [ ] 表情
  - [ ] 特效 (子弹击中/全息)
- [ ] option 选择支

## 基础功能

- [ ] continue
- [ ] st (可以出现在任意位置的文字)
- [ ] stm (居中的特殊文字)
- [ ] clearSt (清空 st 效果, 因为 st 效果是叠加的)
- [ ] all (对所有对象操作)
- [ ] showMenu/hideMenu (显示/隐藏菜单)
- [ ] bgShake (背景抖动)
- [ ] video (弹出视频)
